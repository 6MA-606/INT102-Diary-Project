- Other CSS -
CSS that do not be covered in class
Transform
.example { transform: value }
ใช้เพื่อสร้างการเปลี่ยนแปลงให้กับ Element ยกตัวอย่าง เช่น การเลื่อนตำแหน่ง การปรับสเกล และการหมุน
Clip-path
.example { clip-path: value }
ใช้เพื่อกำหนดการครอบตัด Element นั้น ๆ
Object-fit
.example { object-fit: value }
ใช้เพื่อกำหนดว่าจะให้ element นั้น เติมเต็ม parent element อย่างไร (Ex. cover, contain)
@keyframes
@keyframes indentifier {
 from { opacity: 0% }
 to { opacity: 100% }
}
ใช้เพื่อกำหนดคีย์เฟรมสำหรับใช้กับ CSS animation
Animation
.test { animation: name 1s }
ใช้เพื่อ set ให้ element นั้น ๆ แสดง Animation ด้วย keyframes ที่สร้างไว้ตามรูปแบบที่ต้องการ
Filter
.test { filter: value }
ใช้เพื่อกำหนดฟิลเตอร์ที่จะนำมาซ้อนกับ Element นั้น ๆ
