- Other CSS -
CSS that do not be covered in class
Transform
.example { transform: scale(1.1) }
ใช้เพื่อสร้างการเปลี่ยนแปลงให้กับ Element ยกตัวอย่าง เช่น การเลื่อนตำแหน่ง การปรับสเกล และการหมุน
Clip-path
.example { clip-path: circle() }
ใช้เพื่อกำหนดการครอบตัด Element นั้น ๆ
Object-fit
.example { object-fit: cover }
ใช้เพื่อกำหนดว่าจะให้ element นั้น เติมเต็ม parent element อย่างไร (Ex. cover, contain)
Keyframes
@keyframes indentifier {
 from { opacity: 0% }
 to { opacity: 100% }
}
ใช้เพื่อกำหนดคีย์เฟรมสำหรับใช้กับ CSS animation
Animation
.example { animation: name 1s }
ใช้เพื่อ set ให้ element นั้น ๆ แสดง Animation ด้วย keyframes ที่สร้างไว้ตามรูปแบบที่ต้องการ
Filter
.example { filter: invert() }
ใช้เพื่อกำหนดฟิลเตอร์ที่จะนำมาซ้อนกับ Element นั้น ๆ
Transform-style
.ex { transform-style: preserve-3d }
ใช้เพื่อกำหนดว่า Element นั้น ๆ จะ transform แบบ 2D หรือ 3D
Cursor
.example { cursor: pointer }
ใช้เพื่อหน้าตาของ cursor ที่ hover บน element นั้น ๆ
Transition
.example { transition: .25s }
ใช้เพื่อกำหนดว่าเมื่อ element นั้น ๆ มีเปลี่ยนแปลง จะใช้เวลาเท่าไรในการเปลี่ยนจะรูปแบบปัจจุบันไปเป็นรูปแบบใหม่
